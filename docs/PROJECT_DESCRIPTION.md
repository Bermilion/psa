# MarineService Manager

> Система управления судоремонтными работами: документооборот, персонал, коммуникации, учёт документов.

## 🎯 Цель проекта

Создание централизованной веб-платформы для автоматизации документооборота, управления персоналом, заказчиками и внутренними коммуникациями в судоремонтной компании. Проект ориентирован на повышение прозрачности, контроля сроков и снижения рутины при оформлении технической документации.

---

## 📦 Основные модули

### 1. Документооборот
Создание и управление стандартными и свободными формами отчётов:

- Акт дефектации
- Приложение к акту дефектации
- Техническое задание (ТЗ)
- Технический протокол
- Приложение к техническому протоколу
- Исполнительная ремонтная ведомость
- Акты свободной формы (замеры, опрессовка, освидетельствование и др.)

**Функции:**
- Шаблоны с подстановкой данных (судно, заказчик, дата, ответственные)
- Редактирование полей по шаблону
- Экспорт в PDF
- Подпись (цифровая или имитация)
- Статусы: "Черновик", "На согласовании", "Утверждён", "Архив"

---

### 2. Сводная таблица отчётов
Гибкий интерфейс анализа всех созданных документов.

**Фильтрация по:**
- Статусу
- Заказчику
- Судну
- Периоду (дата создания/утверждения)
- Ответственному лицу

**Гибкие настройки отображения:**
- Выбор видимых колонок
- Сохранение пользовательских профилей отображения
- Экспорт в Excel/CSV

---

### 3. Учёт действующих документов
Реестр обязательных к актуальности документов с контролем сроков.

**Типы документов:**
- Регистровые (Росморречфлот, РМРС и др.)
- Охрана труда (инструктажи, допуски)
- Пожарная безопасность
- Журналы проверки электроинструмента
- Сертификаты персонала (сварка, электромонтаж и др.)

**Функции:**
- Загрузка сканов/фото
- Установка срока действия
- Автоматические уведомления о приближающейся просрочке (за 7, 3, 1 день)
- Календарь напоминаний

---

### 4. Управление персоналом
Центральное хранение данных о сотрудниках.

**Функции:**
- Сводная таблица всех сотрудников
- Форма добавления/редактирования
- Карточка сотрудника:
  - ФИО, должность, специальность
  - Контакты (телефон, email)
  - Персональные данные: паспорт, СНИЛС, ИНН
  - Фото
  - Документы: дипломы, удостоверения, сертификаты (загрузка)
- Возможность поиска по ФИО, специальности, документам

**Доступ:**
- Админ и менеджер — полный доступ
- Сотрудник — только просмотр своей карточки и редактирование контактных данных (по настройке)

---

### 5. Внутренний мессенджер
Коммуникационный модуль, аналогичный Telegram.

**Функции:**
- Групповые чаты по судам, проектам, бригадам
- Личные сообщения
- Отправка сообщений: текст, фото, видео, аудио (в т.ч. голосовые)
- Редактирование и удаление своих сообщений
- Уведомления (веб и email)
- Архив чатов

**Ограничения:**
- Сотрудники не могут создавать или удалять чаты
- Только администраторы и менеджеры управляют составом групп

---

### 6. Управление заказчиками
Реестр контрагентов.

**Функции:**
- Список всех заказчиков
- Форма добавления/редактирования
- Данные: название, ИНН, контакты, юр. адрес, банковские реквизиты
- Привязка к выполненным работам и отчётам

---

## 👥 Роли и права доступа

Используется система на базе **Spatie Laravel Permission**.

| Роль | Доступ |
|------|--------|
| **Администратор** | Полный доступ ко всем разделам: создание, редактирование, удаление. Управление ролями, пользователями, настройками системы. |
| **Менеджер** | Практически полный доступ, кроме системных настроек. Может создавать отчёты, управлять сотрудниками, заказчиками, чатами. |
| **Сотрудник** (механик, сварщик и др.) | <ul><li>Просмотр документов (ОТ, ПБ, инструменты) — только чтение</li><li>Просмотр своей карточки + редактирование контактов</li><li>Доступ к мессенджеру: отправка, редактирование, удаление своих сообщений</li><li>Нет доступа к созданию отчётов (по умолчанию)</li></ul> |

## 🛠️ Технологический стек (Laravel 12)

### Основные пакеты:

| Назначение | Пакет | Описание |
|----------|-------|--------|
| Авторизация и ACL | [`spatie/laravel-permission`](https://spatie.be/docs/laravel-permission) | Роли и разрешения |
| Админка (опционально) | [`laravel-adminpanel`](https://github.com/tallstackui/adminpanel) или кастомная | Быстрый UI для управления |
| PDF-генерация | [`barryvdh/laravel-dompdf`](https://github.com/barryvdh/laravel-dompdf) | Создание PDF из Blade-шаблонов |
| Файловое хранилище | `Laravel Storage` + `S3` или `local` | Хранение сканов, сертификатов, медиа |
| Миграции и сиды | `Laravel Migrations` + `Seeders` | Управление структурой БД |
| API (если нужно) | `Laravel Sanctum` | Токены для SPA или мобильного клиента |
| Уведомления | `Laravel Notifications` | Email/SMS/Database-уведомления о сроках |
| Поиск и фильтрация | `Laravel Scout` + `Meilisearch` (или просто Eloquent) | Гибкий поиск по сотрудникам, документам |
| Frontend | `Livewire 3` + `Alpine.js` + `Tailwind CSS` | Интерактивный UI без сложного JS-фреймворка |
| Чат | Реализация на Livewire + Broadcasting (Pusher / Soketi) | Реальное время в мессенджере |
